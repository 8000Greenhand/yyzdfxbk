# 分析结果

## 1. 原视频内容分析

### 视频价值一句话

帮自媒体创作者把内容生产拆成选题、脚本、包装、发布、复盘几个固定 Skill，减少每次从零开始的混乱感。

### 内容总结

这条视频盘点自媒体从选题调研、文案脚本、内容包装到发布复盘的 AI Skill 工具链。

### 标题分析

原始标题：“自媒体人的skill盘点”“从调研到复盘”“一条视频全讲透”。它的强点是完整清单感，不是单个工具推荐，而是承诺把内容生产流程串起来。对自媒体人来说，“从调研到复盘”比“10个工具”更有收藏价值。

### 开头原话

“如果现在让你马上做一条视频，你脑子里的第一步是什么？”

### 开头分析

这个开头是提问式自检。它没有先讲工具，而是让观众意识到自己没有稳定流程：不知道先选题、先写脚本、还是先找素材。前几秒的价值在于制造“我好像确实没有方法”的轻微焦虑，再顺势引出 Skill 工具链。

### 内容结构分析

1. 用“做视频第一步是什么”切入流程混乱。
2. 讲选题调研：好选题来自数据、爆款和评论区反馈。
3. 讲文案脚本：标题、前三秒、卖点、转场、互动引导。
4. 讲生产包装：字幕、配音、画面、剪辑、素材调用。
5. 讲发布复盘：看数据、分析评论、更新策略。
6. 扩展风格蒸馏、安全审查、版权检查、内容日历等 Skill。

### 爆点分析

这条内容的爆点不是某个具体工具，而是“把混乱的创作过程变成一张流程图”。它很容易被收藏，因为创作者缺的不是一个工具名，而是从选题到复盘的顺序感。

它的传播动机有三个：第一，流程焦虑，很多人做内容靠灵感，看到全流程会想收藏；第二，工具期待，标题里有 Skill，观众会期待拿到可直接用的清单；第三，评论引导空间大，文档、工具、Skill 怎么写都容易引发追问。对你来说，最值得拿走的是“把工作拆成 Skill”，而不是照搬自媒体人的工具清单。

## 2. 工具 / 技术 / 资源

- **选题调研 Skill**
  - 作用：根据账号方向和目标人群，整理用户问题、同行选题和可做角度。
  - 对你有什么用：可以用于判断刷到的视频能不能转成你自己的小红书选题。

- **爆款拆解 Skill**
  - 作用：拆标题、开头、结构、收藏点、评论点。
  - 对你有什么用：适合处理低粉高赞素材，沉淀可迁移的结构。

- **评论区选题 Skill**
  - 作用：从高频评论、提问和争议点里提炼下一条内容。
  - 对你有什么用：后续如果能抓评论区，可以反哺日更选题。

- **反AI味文案检查 Skill**
  - 作用：检查 AI 稿里的模板感、空话、过度升华。
  - 对你有什么用：放在发布前，避免内容看起来太像 AI。

- **发布复盘 Skill**
  - 作用：根据阅读、点赞、收藏、评论等数据判断问题和下一步测试方向。
  - 对你有什么用：用于你账号前三条和后续日更复盘。

## 3. 给我的内容改造方案

### 方案 A：别再收藏AI工具了，打工人真正该先做这5个工作流

- **可改造方向**：不照搬“自媒体 Skill 盘点”，改成普通职场人能用的 AI 工作流清单。
- **标题备选**：
  1. 别再收藏AI工具了，打工人真正该先做这5个工作流
  2. 普通职场人用AI，先别学复杂的，先自动化这5件事
  3. 我发现AI提效最有用的不是写文案，而是做工作流
- **封面文案**：打工人AI提效｜先做5个工作流
- **开头口播**：很多人学AI的第一步是收藏工具，但我后来发现，真正提效的不是工具，而是把你每天重复做的事变成固定流程。
- **内容结构**：收藏工具没用 → 工作流比工具重要 → 5个工作流：反AI味检查、标题封面生成、爆款拆解入库、工作资料沉淀、重复表格处理 → 每个说输入输出 → 评论区问大家最想自动化什么。

### 方案 B：我把小红书日更拆成了几个 AI Skill

- **可改造方向**：展示你如何把自己的日更流程拆成模块，而不是每天临时想。
- **标题备选**：
  1. 我把小红书日更拆成了这几个AI Skill
  2. 做小红书别只问AI写稿，先把流程拆开
  3. 我终于不用每天从0开始想内容了
- **封面文案**：小红书日更｜拆成AI Skill
- **开头口播**：我以前以为用AI做小红书，就是让它帮我写文案。后来发现这样很难稳定，真正有用的是把每一步拆成 Skill。
- **内容结构**：选题判断 → 标题封面 → 正文结构 → 去AI味检查 → 发布复盘 → 评论区选题。
