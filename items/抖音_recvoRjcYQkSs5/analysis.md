# 分析结果

## 1. 原视频内容分析

### 内容总结

基于 GitHub 爆火的黑白极简插画风格，用 AI 做出自己的视觉资产，并长期用于文章、网站和内容配图。

### 标题分析

标题借用了“GitHub 爆火小黑插画”这个热点，再加上“我用 AI 做了自己的版本”，不是单纯搬运热点，而是强调个人化改造。它适合吸引对 AI 视觉、个人IP、内容配图感兴趣的人。

### 开头分析

开头先展示一个已经火起来的视觉风格，再说自己做了一个版本。视觉类内容的优势是结果可以直接被看见，比纯文字教程更容易让观众停留。

### 内容结构分析

1. 先说明 GitHub 小黑插画很火，适合文章和网站配图。
2. 参考黑白极简风格，但调整线条和视觉冲击力。
3. 用 Imagegen 生成自己的角色形象。
4. 压缩图片并上传对象存储，方便后续调用。
5. 用多 agent 批量生成文章配图。
6. 放进知识库和网站首页，形成长期复用的视觉语言。

### 爆点分析

- **点击点**：GitHub爆火、AI做自己的版本、视觉结果可见。
- **收藏点**：流程可复刻，适合做个人视觉资产。
- **评论点**：提示词怎么写、风格怎么稳定、对象存储怎么用、多agent怎么跑。
- **真正有价值的核心**：不是生成一张图，而是把一种风格沉淀成能长期调用的视觉资产库。

## 2. 工具 / 技术 / 资源

- **GitHub 小黑插画风格**：原始视觉参考，特点是黑白极简、线条简单、识别度强。
- **Imagegen**：用于生成个人化角色形象和配图。
- **对象存储**：用于存放压缩后的图片资源，方便后续调用。
- **多 agent 批量配图**：让不同文章或内容批量生成同一风格的插画。
- **视觉语言资产化**：把单张图升级成长期可复用的封面、配图、内页元素。

### 方案 A：我用 AI 做了一套自己的小红书封面模板

- **可改造方向**：把原视频的“个人IP视觉语言”改成你的图文工作台和小红书封面生产场景。
- **标题备选**：
  1. 我终于不用每次重新设计小红书封面了
  2. 我用AI做了一套自己的小红书封面模板
  3. 别再临时找图了，先做一套自己的视觉资产
- **封面文案**：
  - 小红书封面
  - AI视觉模板
- **开头口播**：
  我以前每次发图文都要重新想封面，后来发现真正省时间的不是让 AI 临时画图，而是先做一套能长期复用的视觉模板。
- **内容结构**：封面痛点 → 什么是视觉资产 → 固定背景/字体/信息层级 → 接入本地作图工作台 → 模板化前后对比。

### 方案 B：AI做图真正省时间的地方，不是画得多好看

- **可改造方向**：讲“AI视觉资产库”对普通职场人的价值，不局限于自媒体。
- **标题备选**：
  1. AI做图真正省时间的地方，不是画得多好看
  2. 打工人别只让AI画图，先让它帮你固定模板
  3. 我发现AI做图最有用的地方，是统一风格
- **封面文案**：
  - AI做图
  - 别只追好看
- **开头口播**：
  很多人用AI画图，第一反应是追求好看。但对职场人来说，更重要的是稳定、统一、可复用。
- **内容结构**：追求单图好看的问题 → 什么是视觉资产 → 职场场景应用 → 最小版本怎么做 → 如何接入工作台。
