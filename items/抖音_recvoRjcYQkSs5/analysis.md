1. 开头热点：
作者先借 GitHub 爆火“小黑插画”切入，说明这个黑白极简风格很适合文章、网站和内容配图。

作者不是直接照搬原风格，而是基于黑白极简特点，调整线条和视觉冲击力，做出自己的 01MVP 版本。

核心步骤是用 Imagegen 生成自己的角色形象，让插画不只是单张图，而是形成可以反复出现的视觉资产。

4. 素材管理：
生成后的图片会压缩并上传对象存储，方便后续在文章、网站、知识库中调用。

5. 批量生产：

6. 长期复用：
最后把插画放进知识库和网站首页，形成长期可复用的个人视觉语言。

## 7. 给用户的内容改造方案

### 方案 A：我给自己的小红书做了一套AI封面视觉模板

- 主题：把单次做图升级成长期复用的视觉资产库。
- 标题备选：
  1. 我终于不用每次重新设计小红书封面了
  2. 我用AI做了一套自己的小红书封面模板
  3. 别再临时找图了，先做一套自己的视觉资产
- 封面文案：小红书封面｜AI视觉模板
- 开头钩子：我以前每次发图文都要重新想封面，后来发现真正省时间的不是让AI临时画图，而是先做一套能长期复用的视觉模板。
- 内容结构：
  1. 讲痛点：每次封面都从零开始；
  2. 说明视觉资产的概念：固定背景、固定字体、固定信息层级；
  3. 展示如何用 AI 生成几类背景/插画；
  4. 展示模板如何进入你的本地工作台；
  5. 对比：临时做图 vs 模板化出图。
- 需要准备的材料：你现有小红书封面截图、AI生成底图、模板化前后对比、本地作图工作台录屏。
- 改造逻辑：借用原视频“热门视觉风格变成长期资产”的优势，但改成你更真实的图文工作台和小红书封面生产场景。

### 方案 B：普通职场人也需要自己的AI视觉资产库吗？

- 主题：把视觉资产从个人IP讲到职场表达，如汇报图、图文封面、项目说明图。
- 标题备选：
  1. 我发现AI做图最有用的不是画得多好看
  2. AI做图真正省时间的地方，是统一风格
  3. 打工人别只让AI画图，先让它帮你固定模板
- 封面文案：AI做图｜别只追好看
- 开头钩子：很多人用AI画图，第一反应是追求好看。但对职场人来说，更重要的是稳定、统一、可复用。
- 内容结构：追求单图好看的问题 → 什么是视觉资产 → 职场场景中的应用 → 怎么做最小版本 → 如何接入工作台。
- 需要准备的材料：汇报页/小红书图文/工作台界面三类示例。
- 改造逻辑：将原视频的个人IP视觉语言，迁移到普通职场人的“统一表达模板”。

## 8. Skill / 工作流沉淀

- 名称：小红书封面视觉模板 Skill
- 来源：来自原视频 + 用户场景改造
- 解决的问题：每次发图文都要重新设计封面，风格不统一，耗时高。
- 输入：主题、标题、账号风格、版式要求、颜色偏好、是否需要插画。
- 输出：封面背景建议、插画风格提示词、标题排版建议、可复用模板说明。
- 适合场景：小红书图文、AI工作台展示、职场提效类内容封面。
- 如何用于用户的小红书账号、工作台或实际工作：接入你本地半自动作图工作台，减少每次封面设计成本。

- 名称：个人视觉资产库工作流
- 来源：来自原视频
- 解决的问题：视觉素材分散，无法长期复用。
- 输入：角色风格、背景风格、常用场景、账号定位。
- 输出：封面底图、角色插画、内页装饰元素、素材分类目录。
- 适合场景：小红书账号长期运营、网站文章配图、知识库视觉统一。
- 如何用于用户的小红书账号、工作台或实际工作：把生成的底图和插画归档到固定文件夹，由作图工作台调用。
